Desde la página Plugins puedes importar, administrar y eliminar plugins de Dispatcharr. Las descripciones de los plugins están disponibles en [discord](https://discord.gg/Sp45V5BcxU)

## Mis plugins
* Presiona el botón "Import Plugin" para agregar un plugin, o carga uno en `/data/plugins` y recarga.
* Elimina un plugin haciendo clic en el botón <i data-lucide="trash-2" style="color: Pink; width: 18px;"></i> Delete plugin
* Habilita/deshabilita un plugin haciendo clic en el ícono Toggle a la derecha del botón Delete plugin
!!! info "Información"
    ¿Te interesa el desarrollo de plugins? Consulta la documentación de desarrollo de plugins en el [dispatcharr github](https://github.com/Dispatcharr/Dispatcharr/blob/main/Plugins.md)

## Buscar plugins
Los usuarios administradores pueden explorar, instalar y actualizar plugins directamente desde sus repositorios fuente mediante el Plugin Hub. El repository oficial de Dispatcharr Plugins viene preconfigurado

* Busca, ordena y filtra plugins en la parte superior de la página
* Usa el botón "Manage Repos" en la parte superior de la página para agregar repositorios de plugins mediante manifest URL
    * Refresh Interval - Define el intervalo de actualización para todos los repositorios (usa 0 para deshabilitarlo)
    * Se admiten repos de terceros con una clave pública GPG opcional
