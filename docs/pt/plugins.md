Na página de plugins você pode importar, gerenciar e excluir plugins do Dispatcharr. Descrições dos plugins estão disponíveis no [discord](https://discord.gg/Sp45V5BcxU)

## Meus Plugins
* Pressione o botão "Import Plugin" para adicionar um plugin, ou carregue um em `/data/plugins` e recarregue.
* Remova um plugin clicando no botão <i data-lucide="trash-2" style="color: Pink; width: 18px;"></i> Excluir plugin
* Ative/desative um plugin clicando no ícone Toggle à direita do botão Excluir plugin
!!! info
    Interessado no desenvolvimento de plugins? Consulte a documentação de desenvolvimento de plugins no [github do Dispatcharr](https://github.com/Dispatcharr/Dispatcharr/blob/main/Plugins.md) 

## Encontrar Plugins
Usuários administradores podem navegar, instalar e atualizar plugins diretamente de seus repositórios de origem via Plugin Hub. O repositório oficial de Plugins do Dispatcharr vem pré-configurado

* Pesquise, ordene e filtre plugins no topo da página
* Use o botão "Manage Repos" no topo da página para adicionar repositórios de plugins via URL de manifesto
    * Refresh Interval - Defina o intervalo de atualização para todos os repositórios (defina 0 para desativar)
    * Repositórios de terceiros são suportados com chave pública GPG opcional
