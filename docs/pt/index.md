## Visão Geral
O Dispatcharr ajuda você a...

- Coletar, organizar e distribuir seus streams IPTV.
- Gerenciar dados de EPG (Guia Eletrônico de Programação).
- Realizar filtragem avançada, importação de playlists (M3U) e retransmissão.
- Oferecer uma interface amigável para gerenciar seus streams, tudo a partir de uma solução auto-hospedada.

O Dispatcharr é um proxy e editor de playlists IPTV (m3u m3u8). É um aplicativo proxy de IPTV que permite aos usuários organizar listas M3U complexas, gerenciá-las e retransmiti-las para diversas aplicações como Emby, Plex, Jellyfin ou qualquer outra plataforma de streaming compatível. Ele oferece a flexibilidade de importar um número ilimitado de arquivos M3U e EPG, suportando uma ampla variedade de conteúdo de streaming. O Dispatcharr oferece streaming rápido e confiável em múltiplos sistemas operacionais.

Recursos adicionais, como proxy robusto e suporte para múltiplos clientes em um único stream de origem, tornam o Dispatcharr uma solução completa para as necessidades atuais de streaming.

  - Motor Proxy de Streaming - Stream mais estável, com menos conexões ao provedor e troca automática de stream quando buffering é detectado
  - Painel de Estatísticas em Tempo Real - Monitore a saúde dos streams e a atividade dos clientes ao vivo.
  - EPG Auto-Match - Mapeia automaticamente os dados do guia para seus canais
  - Compatibilidade de Saída - Funciona com Plex, Jellyfin, Emby, ChannelsDVR e mais (HDHomeRun, M3U, XMLTV, Xtream Codes API)
