A página Guia de TV exibirá por padrão os canais para os quais há dados de EPG carregados.

* Você pode pesquisar por nome do canal e/ou filtrar por grupos e perfis de canais
* Clique no botão `Series Rules` para gerenciar gravações recorrentes de Séries
    * Qualquer programa pode ser considerado uma série (incluindo títulos que são filmes)
* Clique no logo de um canal para pré-visualizá-lo.
* Clique nas entradas do guia para expandir os dados do guia, pré-visualizar o canal ou agendar gravação
    * Você pode agendar a gravação com as seguintes opções:
        * Apenas este episódio
        * Todos os episódios
        * Apenas novos episódios (seus dados de EPG devem incluir tags `<new />` para identificar novos episódios)
!!! note
    Você pode gravar por canal e horário a partir da página DVR
